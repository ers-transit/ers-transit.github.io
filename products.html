Available Data Products

One of the goals of the ERS program is to provide the community with data products from the ERS observations. Each result which has been published thus far has an affiliated Zenodo repository containing those data products, including stellar spectra, white-light curves, spectroscopic light curves, transmission spectra, and model fits. The full ERS Zenodo community can be found here.

We link all of the current publications and data products here as well, sorted by observation. Publication titles link to the data products; references link to the published manuscript.
